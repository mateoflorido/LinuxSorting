## Installation

Use CMake to create makefiles and build the binaries.

```bash
mkdir build
cd build
cmake ..
make
```

## Usage

Use *csorth, csortp or csortpexec* to run every program.
```bash
./csorth examplefile.txt
```
Sample files for testing are included on src/TestFiles
